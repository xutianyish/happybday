# Happy Birthday Card

## Getting Started
 * Clone the repo
 ```
 git clone https://github.com/xutianyish/happybday.git
 ```


## Deploy the app using Heroku
 * create heroku project
 ```
 git clone https://github.com/xutianyish/happybday.git
 ```
 * Adding files
 ```
 git add -A
 ```
 * commiting the change
 ```
 git commit -m "commiting to project"
 ```
 * push to github
 ```
 git push
 ```
 * push to heroku repo
 ```
 git push heroku master
 ```
